# 1.6.2 vue cli3 配置省略文件扩展名

```
目前项目中引入组件，如 

import vueOfficeHeader from '@/components/vue-office-header';

提示找不到文件

加上扩展名就可以了

import vueOfficeHeader from '@/components/vue-office-header.vue';

```

>看到issue 回复

```
将完全停止支持无扩展名的单个文件组件的导入（当前仅存在类型检查错误）
```
