# 1.5.7 vue中关闭定时器

>有人说是加window.setInterval，有人是 window.clearInterval

```
mounted() {
    const that = this;
    that.syncTime = null;
    that.syncTime = window.setInterval(() => {
      that.getSyncStatus();
    }, 3000);
    // 如果不加监听，跳转页面后定时器还会执行
    that.$once('hook:beforeDestroy', () => {
      clearInterval(timer);
    });
},

beforeDestroy() {
    const that = this;
    window.clearInterval(that.syncTime);
    that.syncTime = null;
  },
```
