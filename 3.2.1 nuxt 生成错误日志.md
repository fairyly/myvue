# 3.2.1 nuxt 生成错误日志


## nuxt-winston-log

>基于 winston

## install
```
 yarn add nuxt-winston-log # or npm i nuxt-winston-log
```

## usage

```
#nuxt.config.js

{
  modules: ['nuxt-winston-log']
}
```
