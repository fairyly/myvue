# 1.2.6 vue meta管理头部标签


## Installation
```
Yarn
$ yarn add vue-meta
npm
$ npm install vue-meta --save
```

## Usage
```
See the documentation for more information

import Vue from 'vue'
import VueMeta from 'vue-meta'

Vue.use(VueMeta, {
  // optional pluginOptions
  refreshOnceOnNavigation: true
})
```
